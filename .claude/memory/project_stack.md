Kanary Boards is a kanban board app. Key stack facts:

**Runtime:** Deno 2.x (not Node.js). Package manager is `deno install`. Tasks run via `deno run --allow-all astro <cmd>`. Scripts defined in `deno.jsonc`. Unstable APIs enabled via `"unstable": ["kv"]` in deno.jsonc.

**Framework:** Astro 6.x with the Deno adapter (`@deno/astro-adapter`). SSR mode (`output: "server"`). Deploys to Deno Deploy. Pages: `src/pages/index.astro` (board), `src/pages/task/[taskId].astro` (task deep-link with SEO meta tags). Shared layout at `src/layouts/BaseLayout.astro`. Middleware at `src/middleware.ts` (sets `boardId` cookie and `Astro.locals.boardId`).

**Routing:** react-router-dom 7.x inside the React island. `BrowserRouter` wraps `BoardWrapper`. Routes: `/` (main board) and `/task/:taskId` (board + edit modal open). Astro handles SSR for both; react-router handles client-side navigation.

**UI:** React 19. Components live in `src/components/`. State is managed with Context API + useReducer (no Redux/Zustand). Three contexts: `BoardStateContext`, `BoardDispatchContext`, `BoardRefsContext`. Icons via `astro-icon` + `@iconify/json` + `@iconify/tailwind4`. Rich text via `@lyfie/luthor`.

**Styling:** Tailwind CSS 4.x + DaisyUI 5.x. Uses `className=` (React convention). Two themes: `kanary-day` (light) and `kanary-night` (dark), toggled via ThemeToggle.tsx.

**AI:** Google GenAI SDK (`@google/genai`) via a POST endpoint at `src/pages/api/generate-tasks.ts`. Generates task titles and checklist items.

**Persistence:** Deno KV (migrated from localStorage). KV utilities in `src/lib/kv.ts`. Board stored at `["board", boardId]`; task titles indexed at `["task_meta", taskId]` for SSR SEO lookups. API routes at `src/pages/api/board.ts` (GET/PUT/DELETE). BoardContext fetches on mount, saves with 500ms debounce, and auto-migrates any existing localStorage data to KV on first load.

**boardId:** A UUID generated by Astro middleware on first visit, stored as a cookie (`boardId`), available server-side as `Astro.locals.boardId` and passed as a prop to the React island. Designed for future multi-user support.

**Type checking:** `deno run --allow-all astro check` — passes clean with exit 0.

**Lock file:** `deno.lock` (do not manually edit).
